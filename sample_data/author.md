# 芥川龍之介 作品一覧

## 作家情報

| 項目 | 内容 |
|------|------|
| 名前 | 芥川龍之介（あくたがわ りゅうのすけ） |
| 生没年 | 1892年 - 1927年 |
| 出身地 | 東京府東京市 |
| 代表作 | 羅生門、蜘蛛の糸、鼻 |

## 主要作品

### 羅生門（1915年）

平安時代末期の京都を舞台にした短編小説。人間の利己心と生存本能を描いた作品。

### 蜘蛛の糸（1918年）

お釈迦様と罪人カンダタの物語。児童文学としても知られる。

### 鼻（1916年）

長い鼻を持つ僧侶の物語。夏目漱石に絶賛された出世作。

### 杜子春（1920年）

中国の伝奇に基づく物語。人間性の本質を問う作品。

## 文学的特徴

- 古典文学や説話を題材にした作品が多い
- 簡潔で美しい文体
- 人間心理への深い洞察
- 皮肉とユーモアを含んだ作風
